## An Idea

Scrolling through various articles on my 'phone a day or so ago, I 
noticed that the browser hides the address bar most of the time and 
only shows it when I actually need it.  That makes really good use 
of the small screen, and I wondered if a blog layout with the same
behaviour was possible.  
The good news: it is.  
The even better news:  it is easy.

## The steps
Using cascading style sheets, my plan is:
- Canvas that takes up the whole screen.
- Central column for the content.
- Solid block at the bottom for the navigation.
- Padded central column in the navigation for a ruler and the links.
- Javascript to move the navigation onto or off of the screen.
- Padded header with a ruler.
- Padded main content.
- Simplify presentation of some elements.

### Cascading style sheet (CSS)
This is what I came up with.  It's quite spartan, so I'm thinking that
I'll call it the Spartan Theme.  Or maybe the Epicurean Theme.  The
ancient Greek philosopher Epicurus taught that happiness is achieved 
through simple living, moderation, friendship, and the avoidance of 
pain and unnecessary desires. The richest life is light on its feet 
and contentment comes from needing less, rather than acquiring more.
```
* { box-sizing: border-box; }
html, body { margin: 0; background-color: #fff; 
  color: #888; text-align: justify; }
/* Add a container that centers a block, still with */
/* no margins                                       */
.container, .nav-content { max-width: 650px; margin: 0 auto; 
  padding: 0; width: 100%; }
/* Add a bottom navigation bar on top, also with no */
/* margins                                          */
.nav-bottom { position: fixed; bottom: 0; width: 100%; 
  background-color: #fff; transition: transform 0.3s ease; 
  z-index: 1000; }
/* Put content in the navigation bar, with padding  */
.nav-content { margin-top: 4px; border-top: 1px solid #bbb; 
  padding: 10px; padding-bottom: 10px; height: 100%; display: flex; 
justify-content: center; gap: 5px 30px; align-items: center; 
  list-style: none; flex-wrap: wrap; }
/* Move the navigation bar out of the way and use   */
/* javascript to reveal it when appropriate.        */
.nav-hidden { transform: translateY(100%); }
/* Put content in the header, with the same padding */
header { border-bottom: 1px solid #bbb; margin-bottom: 4px; 
  padding: 10px; }
/* Put content in the main block, with extra padding */
/* on the right for the scroll bar and on the bottom */
/* to fill up space.	                             */
main { padding: 0 12px 100px 10px; }
/* Take the bullet and padding off of lists */
ul { list-style-type: none; padding: 0; }
/* Simplify presentation of links */
a { text-decoration: none; color: inherit; }
a:hover { text-decoration: underline; }


```
### Javascript
It's a very simple script that hides the navigation bar when 
scrolling down and reveals it when scrolling up, but I needed a
bit of AI help with it.
```
let lastScrollTop = 0;
const navbar = document.querySelector('.nav-bottom');
window.addEventListener('scroll', function() {
    let scrollTop = window.pageYOffset || document.documentElement.scrollTop;
    // Guard against "bounce" scrolling (common on mobile)
    if (scrollTop <= 0) {
        navbar.classList.remove('nav-hidden');
        return;
    }
    if (scrollTop > lastScrollTop) {
        navbar.classList.add('nav-hidden'); // Scrolling down
    } else {
        navbar.classList.remove('nav-hidden'); // Scrolling up
    }
    lastScrollTop = scrollTop;
}, false);
```

I think the Epicurean Theme looks good.  It's more minimal than
